# Traffic_Light_Controller_Verilog
# 🚦 Traffic Light Controller for a T-Junction using Verilog HDL

A Traffic Light Controller designed using **Verilog HDL** and simulated in **Xilinx Vivado**. The controller uses a **Finite State Machine (FSM)** to control traffic signals at a **T-junction (three-road intersection)** with predefined timing sequences.

## 🛠️ Tools Used

- Verilog HDL
- Xilinx Vivado
- Vivado Simulator

## ✨ Features

- Finite State Machine (FSM) based design
- Controls traffic lights for a **T-junction**
- Fixed timing sequence for Green, Yellow, and Red signals
- Simulated and synthesized using Vivado
- Low FPGA resource utilization
